# [USACO08OCT]Building A Fence G

## 题目描述

请问他有多少种不同的切割方式能使切割出来的木板围成一个四面的栅栏。 

注意： 

1. 不要考虑对称性的问题，不需要去除对称的方案和类似的复杂问题；
2. 栅栏围成的面积必须大于 0；
3. 结果可以用 32 位整数存储。

## 输入格式

一个整数 $N$。

## 输出格式

Farmer John 能将木板分割开来并能围成四边形的方案数。

## 样例 #1

### 样例输入 #1
```
6
```

### 样例输出 #1

```
6
```

## 提示

Farmer John 有 10 种方法将木板分成四块：
- (1, 1, 1 ,3);
- (1, 1, 2, 2); 
- (1, 1, 3, 1); 
- (1, 2, 1, 2); 
- (1, 2, 2, 1); 
- (1, 3, 1, 1); 
- (2, 1, 1, 2); 
- (2, 1, 2, 1); 
- (2, 2, 1, 1);
- (3, 1, 1, 1)。

其中有四种情况是不能围成一个四边形的：
- (1, 1, 1, 3),
- (1, 1, 3, 1),
- (1, 3, 1, 1),
- (3, 1, 1, 1)。
