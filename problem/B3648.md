# [语言月赛202208] 你几岁了

## 题目描述

小 A 今年 $x$ 岁。他刚学了英语，想告诉其他人：`I am x years old.`

## 输入格式

输入一个数 $x(2\le x \le 100)$，代表小 A 的年龄。

## 输出格式

输出一句话。

## 样例 #1

### 样例输入 #1
```
2
```

### 样例输出 #1

```
I am 2 years old.
```

## 样例 #2

### 样例输入 #2
```
5
```

### 样例输出 #2

```
I am 5 years old.
```

完成的程序的编写后，请记得在 `Dev-c++` 上编译运行，然后测试样例，比较自己的输出和样例输出是不是**一字不差**。
