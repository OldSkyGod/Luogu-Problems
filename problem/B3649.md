# [语言月赛202208] 大小比较

## 题目描述

给出整数 $a,b$，若 $a \le b$，输出 `YE5`，否则输出 `N0`（第二个字符为阿拉伯数字）。


## 输入格式

输入一行两个整数 $a,b$。

## 输出格式

输出一行，为你的答案。

## 样例 #1

### 样例输入 #1
```
3 5
```

### 样例输出 #1

```
YE5
```

## 样例 #2

### 样例输入 #2
```
5 3
```

### 样例输出 #2

```
N0
```

## 提示

对于 $100\%$ 的数据，$0 \le a,b \le 10^5$。
