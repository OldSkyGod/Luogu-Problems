## 题目描述

给定 $n$ 个数，请从中选出若干个数，使得总和为偶数。

请最大化这个总和。

## 输入格式

第一行包含一个正整数 $n$。

第二行包含 $n$ 个正整数$a_1,a_2,...,a_n$。

## 输出格式

输出一个正整数，即最大的总和，若无解则输出 `NIESTETY`。

## 样例 #1

### 样例输入 #1
```
5
2 4 3 3 3```

### 样例输出 #1

```
12```

## 提示

对于 $100\%$ 的数据，$1\le n\le 10^6$，$1\le a_i\le 10^3$。
