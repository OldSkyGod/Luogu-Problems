# 【数学1】小浩的幂次运算

## 题目背景



## 题目描述

幂次运算让小浩觉得有些烦，最近他碰到了一些问题。  
他需要求出所有 $w^i$ 使得 $l \le w^i \le r$ ，于是他找到了你。


## 输入格式

输入为一行，有三个整数 $l,r,w$。


## 输出格式

输出为一行，输出所有满足条件的数字，用空格分开。  
如果没有满足条件的数字，则输出 $-1$。


## 样例 #1

### 样例输入 #1
```
1 10 2```

### 样例输出 #1

```
1 2 4 8```

## 样例 #2

### 样例输入 #2
```
2 4 5```

### 样例输出 #2

```
-1```

## 提示

【数据范围】  
对于 $100\%$ 的数据：$1\le l\le r \le 1 \times 10^{18}$ , $1\le w \le10^9$。
