# 『MdOI R3』Number

## 题目背景



## 题目描述

生活中我们会碰到一些特殊数字，这些数字使用一些特殊表达方式后会方便记忆和使用。比如说，$10^9+7$ 就非常常见——它相比于 $1000000007$，更方便选手看清数字而不必数 $0$，而且没有科学计数法的精度损失。

你现在有一个形如 $10^{k}+x$ 的数字，请还原成一般写法。

## 输入格式

一行两个整数 $k,x$。

## 输出格式

一行一个整数表示 $10^k+x$。

## 样例 #1

### 样例输入 #1
```
9 7```

### 样例输出 #1

```
1000000007```

## 提示

【样例解释】   

$10^9+7=1000000007$。
