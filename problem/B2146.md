# Hermite 多项式

## 题目描述

求 Hermite 多项式的值。

$$
h_n(x)=\left\{
\begin{aligned}
1&，&n=0\\
2x&，&n=1\\
2xh_{n-1}(x)-2(n-1)h_{n-2}(x)&，& n>1
\end{aligned}
\right.
$$



对给定的 $x$ 和正整数 $n$，求多项式的值。

## 输入格式

输入 $n$ 和 $x$ 。

## 输出格式

求多项式的值。

## 样例 #1

### 样例输入 #1
```
1 2
```

### 样例输出 #1

```
4
```

## 提示

$(n \le 8，x \le 8)$
