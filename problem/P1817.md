# 棋盘染色

## 题目描述

给定一个 $N \times M$ 的网格，每个格子可以染成黑色或者白色，要求所有黑色格子连通，所有白色格子连通，并且至少有一个黑色格子贴边，至少有一个白色格子贴边。问有多少种染色方法？ 

## 输入格式

第一行有两个正整数 $N, M$。

## 输出格式

只有一个正整数表示答案。


## 样例 #1

### 样例输入 #1
```
1 2
```

### 样例输出 #1

```
2
```

## 样例 #2

### 样例输入 #2
```
2 3
```

### 样例输出 #2

```
30
```

## 提示

对于 $100 \%$ 的数据：$1 \le N \le 7$，$1 \le M \le 8$。
