# 【模板】自适应辛普森法 1

## 题目背景



## 题目描述

试计算积分

$\displaystyle{\int_L^R\frac{cx+d}{ax+b}\mathrm{d}x}$

结果保留至小数点后 $6$ 位。

数据保证计算过程中分母不为 $0$ 且积分能够收敛。

## 输入格式

一行，包含 $6$ 个实数 $a,b,c,d,L,R$。

## 输出格式

一行，积分值，保留至小数点后 $6$ 位。

## 样例 #1

### 样例输入 #1
```
1 2 3 4 5 6```

### 样例输出 #1

```
2.732937```

## 提示

$a,b,c,d\in[-10,10]$，$-100\le L<R\le 100$ 且 $R-L\ge1$。
