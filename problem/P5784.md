# [CQOI2008]矩阵的个数

## 题目背景



## 题目描述

给出一个 $n\times 3$ 的非负整数矩阵的各行各列之和，统计有多少个矩阵满足此条件。输出答案模 $10^{17}$ 的值。

## 输入格式

第一行包含四个正整数 $N$，$c_1$，$c_2$，$c_3$，即行数与三列之和。第二行包含 $N$ 个正整数，即各行三个数之和。每行每列之和均不超过 $125$。


## 输出格式

仅一个数，满足条件的矩阵个数模 $10^{17}$ 的值。

## 样例 #1

### 样例输入 #1
```
3 2 3 4
1 2 6
```

### 样例输出 #1

```
17```
