# 幸运数与排列

## 题目背景



## 题目描述

一个数是幸运数当且仅当这个数仅由 $4$ 和 $7$ 构成，比如 $47$，$744$，$4747$。

询问在 $1$ 到 $n$ 的全排列中字典序第 $k$ 小的排列中，有多少个幸运数在排列中的位置编号也是幸运数。

## 输入格式

一行，包含两个整数 $n$，$k$。

## 输出格式

一个整数表示答案。如果不存在第 $k$ 小的排列，则输出 $-1$。

## 样例 #1

### 样例输入 #1
```
7 4
```

### 样例输出 #1

```
1```

## 样例 #2

### 样例输入 #2
```
4 7
```

### 样例输出 #2

```
1```

## 提示

#### 样例 1 解释

排列为 $1,2,3,4,6,7,5$。

#### 样例 2 解释

排列为：$2,1,3,4$。

#### 数据规模与约定

- 对于 $30\%$ 的数据，保证 $n\le 8$。
- 对于 $60\%$ 的数据，保证 $n\le 100$。
- 对于 $100\%$ 的数据，保证 $1 \leq n,k\le 10^9$。
