# [PA2021] Zbalansowane słowa

## 题目描述

定义一个字符串 $s$ 是好的，当且仅当它包含的所有字符出现次数相同。

好的字符串例如：`mama`、`aabbcbcccbaa`。

不好的字符串例如：`ovo`。

给定一个字符串 $s$，求出 $s$ 好的**子串**数量。

## 输入格式

一行，一个字符串 $s$。

## 输出格式

一行，一个整数，表示所求的值。

## 样例 #1

### 样例输入 #1
```
aabbabcccba
```

### 样例输出 #1

```
28
```

## 提示

对于 $100\%$ 的数据，$1 \leq |s| \leq 3 \times 10^5$，$s$ 中只含 `a`、`b`、`c` 三个小写字母。
