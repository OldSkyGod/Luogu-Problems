# 树的数量

## 题目背景



## 题目描述

求出包含 $n$ 个节点（无标号）的有根 $m$ 叉树的个数，对 $10\,007$ 取模。

两个有根树相同，当且仅当其根节点相同，且**从左到右**每一棵子树也相同。特别地，两个有根树均为空树，视为两个有根树相同。

## 输入格式

输入两个整数 $n$，$m$。


## 输出格式

输出包含 $n$ 个节点（无标号）的有根 $m$ 叉树的个数，对 $10\,007$ 取模后的值。

## 样例 #1

### 样例输入 #1
```
2 2```

### 样例输出 #1

```
2```

## 提示

$n,m \leq 127$
