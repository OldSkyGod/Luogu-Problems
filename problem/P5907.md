# 数列求和加强版 / SPOJ MOON4

## 题目描述

给定 $n,a,k$，求：

$$\sum_{i=1}^n i^k a^i$$
答案对 $998244353$ 取模。

## 输入格式

输入一行三个正整数 $n,a,k$。

## 输出格式

输出一行一个整数表示答案。

## 样例 #1

### 样例输入 #1
```
6 4 3
```

### 样例输出 #1

```
1030980
```

## 样例 #2

### 样例输入 #2
```
233333333 114514 1919810
```

### 样例输出 #2

```
3191086
```

## 提示

【数据范围】    
对于 $20\%$ 的数据，$1\le n,k \le 10^5$。  
对于 $100\%$ 的数据，$1\le k \le 10^7$，$1\le n,a \le 998244352$。
