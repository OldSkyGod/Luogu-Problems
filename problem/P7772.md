# [COCI2009-2010#2] FAKTOR

## 题目描述

给定 $A,I$，求一个最小的 $N$，使得$\lceil\frac{N}{A}\rceil\geq I$。

## 输入格式

一行两个整数 $A,I$。

## 输出格式

一行一个整数 $N$。

## 样例 #1

### 样例输入 #1
```
38 24
```

### 样例输出 #1

```
875
```

## 样例 #2

### 样例输入 #2
```
1 100
```

### 样例输出 #2

```
100
```

## 样例 #3

### 样例输入 #3
```
10 10
```

### 样例输出 #3

```
91
```

## 提示

$1\leq A,I\leq 100$。
