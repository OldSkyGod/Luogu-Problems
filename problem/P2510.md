# [HAOI2008]下落的圆盘

## 题目描述

有n个圆盘从天而降，后面落下的可以盖住前面的。求最后形成的封闭区域的周长。看下面这副图, 所有的红色线条的总长度即为所求.

![](https://cdn.luogu.com.cn/upload/pic/1628.png)


## 输入格式

n ri xi y1 ... rn xn yn


## 输出格式

最后的周长，保留三位小数


## 样例 #1

### 样例输入 #1
```
2
1 0 0
1 1 0
```

### 样例输出 #1

```
10.472
```

## 提示

数据规模

n<=1000
