# 数列求和

## 题目背景



## 题目描述

给定 $n,a,k$，求：

$$\sum_{i=1}^n i^k a^i$$

答案对 $10^9 + 7$ 取模。

## 输入格式

输入一行三个正整数 $n,a,k$。

## 输出格式

输出一行一个整数表示答案。

## 样例 #1

### 样例输入 #1
```
3 4 0```

### 样例输出 #1

```
84```

## 样例 #2

### 样例输入 #2
```
3 10 1```

### 样例输出 #2

```
3210```

## 样例 #3

### 样例输入 #3
```
3 9 2```

### 样例输出 #3

```
6894```

## 提示

| 测试点编号 | $n=$ | $k=$ |
| :----------: | :----------: | :----------: |
| $1$ | $10^6$ | $10^3$ |
| $2$ | $10^6$ | $2*10^3$ |
| $3$ | $10^{18}$ |  $0$|
| $4$ | $10^{18}$ |$1$  |
| $5,6$ |  $10^{18}$| $2$ |
| $7,8$ |  $10^{18}$|  $10^3$|
| $9,10$ |$10^{18}$  |  $2*10^3$|
对于 $100\%$ 的数据，$n\le10^{18}$，$a\le10^9$，$k\le2000$
