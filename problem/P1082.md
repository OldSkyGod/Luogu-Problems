# [NOIP2012 提高组] 同余方程

## 题目背景



## 题目描述

求关于 $ x$ 的同余方程 $ a x \equiv 1 \pmod {b}$ 的最小正整数解。


## 输入格式

一行，包含两个整数 $a,b$，用一个空格隔开。


## 输出格式

一个整数 $x_0$，即最小正整数解。输入数据保证一定有解。


## 样例 #1

### 样例输入 #1
```
3 10```

### 样例输出 #1

```
7```

## 提示

### 数据规模与约定

- 对于 $40%$ 的数据，$2 ≤b≤ 1,000$；
- 对于 $60%$ 的数据，$2 ≤b≤ 50,000,000$；
- 对于 $100%$ 的数据，$2 ≤a, b≤ 2,000,000,000$。
