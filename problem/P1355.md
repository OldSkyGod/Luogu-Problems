# 神秘大三角

## 题目描述

判断一个点与已知三角形的位置关系。


## 输入格式

前三行：每行一个坐标，表示该三角形的三个顶点

第四行：一个点的坐标，试判断该点与前三个点围成三角形的位置关系

(详见样例)

所有坐标值均为整数。


## 输出格式

若点在三角形内（不含边界），输出1；

若点在三角形外（不含边界），输出2；

若点在三角形边界上（不含顶点），输出3；

若点在三角形顶点上，输出4。


## 样例 #1

### 样例输入 #1
```
(0,0)
(3,0)
(0,3)
(1,1)
```

### 样例输出 #1

```
1
```

## 提示

【数据规模与约定】

对于100%数据，0<=所有点的横、纵坐标<=100
