# [BalkanOI2003] Farey 序列

## 题目描述

把所有分子和分母都 $\leq n$ 的**最简真分数**从小到大排成一行，形成的序列称为 Farey 序列。

求出 $n$ 所对应的 Farey 序列中第 $k$ 小的数。

## 输入格式

一行，两个整数 $n, k$。

## 输出格式

一行，两个整数 $p, q$，表示答案 $\frac{p}{q}$。

## 样例 #1

### 样例输入 #1
```
5 6
```

### 样例输出 #1

```
3 5
```

## 提示

对于 $100\%$ 的数据，$2 \leq n \leq 4 \times 10^4$，$1 \leq k \leq$ 符合条件的分数的个数。
