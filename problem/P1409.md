# 骰子

## 题目背景



## 题目描述

$n$ 个人排成一排，你排在第 $m$ 个。

每轮队首的人投一次骰子。

- 若掷到 $1$，则队首的人获胜。
- 若掷到 $2,4,6$，则队首的人排到队尾。
- 若掷到 $3,5$，则队首的人出队。

若队列中仅剩一人，则该人获胜，求你获胜的概率。

## 输入格式

一行两个整数 $n,m$。


## 输出格式

一个实数表示获胜概率，精确到小数点后 $9$ 位。


## 样例 #1

### 样例输入 #1
```
2 1
```

### 样例输出 #1

```
0.444444444
```

## 提示

- 对于 $30\%$ 的数据，$0<m\le n\le 10$。
- 对于 $100\%$ 的数据，$0<m\le n\le 1000$。
