# 火烧赤壁

## 题目描述

给定每个起火部分的起点和终点，请你求出燃烧位置的长度之和。

## 输入格式

第一行一个整数，表示起火的信息条数 $n$。  
接下来 $n$ 行，每行两个整数 $a, b$，表示一个着火位置的起点和终点（**注意：左闭右开**）。

## 输出格式

输出一行一个整数表示答案。

## 样例 #1

### 样例输入 #1
```
3
-1 1
5 11
2 9
```

### 样例输出 #1

```
11
```

## 提示

### 数据规模与约定

对于全部的测试点，保证 $1 \leq n \leq 2 \times 10^4$，$-2^{31} \leq a \leq  b \lt 2^{31}$，且答案小于 $2^{31}$。
