# [JRKSJ R6] 牵连的世界

## 题目描述

给定 $n,m$，求

$$\sum_{i=1}^n \sum_{j=1}^m \sigma_0(ij)\varphi(ij)$$

## 输入格式

两个整数 $n,m$。

## 输出格式

一个整数，表示答案。答案对 $10^9+7$ 取模。

## 样例 #1

### 样例输入 #1
```
5 5
```

### 样例输出 #1

```
453
```

## 样例 #2

### 样例输入 #2
```
20 20
```

### 样例输出 #2

```
173825
```

## 提示

$\sigma_0,\varphi$ 分别为因数个数函数，欧拉函数。

本题可能轻微卡常。

### 数据规模
本题采用捆绑测试。

| $\text{Subtask}$ | $n,m\le$ |  $\text{Score}$ |
| :----------: | :----------: | :----------: |
| $1$ | $10^3$ | $10$ |
| $2$ | $10^5$ |  $30$ |
| $3$ | $2\times 10^5$ | $30$ |
| $4$ | $5\times 10^5$ | $30$ |
| $5$ | $3\times 10^6$ | $1$ |

对于所有数据，$1\le n,m\le 3\times 10^6$。

出于某些原因，你只要得到了 $\ge 100$ 分就可以通过此题。
