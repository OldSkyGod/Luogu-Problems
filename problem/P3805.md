# 【模板】manacher 算法

## 题目描述

给出一个只由小写英文字符 $\texttt a,\texttt b,\texttt c,\ldots\texttt y,\texttt z$ 组成的字符串 $S$ ,求 $S$ 中最长回文串的长度 。

字符串长度为 $n$。

## 输入格式

一行小写英文字符 $\texttt a,\texttt b,\texttt c,\cdots,\texttt y,\texttt z$ 组成的字符串 $S$。

## 输出格式

一个整数表示答案。


## 样例 #1

### 样例输入 #1
```
aaa
```

### 样例输出 #1

```
3
```

## 提示

$1\le n\le 1.1\times 10^7$。
