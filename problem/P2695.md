# 骑士的工作

## 题目描述

每个人都可以砍掉一个大小不超过 $z$ 的头，需要 $z$ 个金币，求最小花费。


## 输入格式

第一行两个整数 $n$，$m$。

下接 $n$ 行，一个整数表示 $n$ 个头的大小。

下接 $m$ 行，每个人可以砍的头大小和需要的金币数。


## 输出格式

一个整数，最小花费。如果无解，输出 `you died!`。


## 样例 #1

### 样例输入 #1
```
2 3
5 
4
7 
8
4
```

### 样例输出 #1

```
11
```

## 提示

对于所有数据，$1 \le n,m \le 2 \times 10^4$。
