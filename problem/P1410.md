# 子序列

## 题目描述

给定一个长度为 $N$（$N$ 为偶数）的序列，问能否将其划分为两个长度为 $N / 2$ 的严格递增子序列。

## 输入格式

若干行，每行表示一组数据。

对于每组数据，首先输入一个整数 $N$，表示序列的长度。之后 $N$ 个整数表示这个序列。

## 输出格式

输出行数与输入行数相同。

对于每组数据，如果存在一种划分，则输出 `Yes!`，否则输出`No!`。

## 样例 #1

### 样例输入 #1
```
6 3 1 4 5 8 7
6 3 2 1 6 5 4
```

### 样例输出 #1

```
Yes!
No!
```

## 提示

**【数据范围】**

共三组数据，每组数据行数<=50，0 <= 输入的所有数 <= 10^9

第一组(30%)：N <= 20

第二组(30%)：N <= 100

第三组(40%)：N <= 2000
