# [COCI2009-2010#2] POSLOZI

## 题目描述



## 输入格式

第一行两个整数 $N,$ $M$。  
第二行 $N$ 个整数，表示排列。  
接下来 $M$ 行，第 $i$ 行有两个整数，表示第 $i$ 种修改方式。

## 输出格式

首行一个整数 $\mathit{ope\_cnt}$，表示最少的修改次数。  
接下来 $\mathit{ope\_cnt}$ 行，每行一个整数，表示进行第 $i$ 种修改。
