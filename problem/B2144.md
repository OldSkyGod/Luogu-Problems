# 阿克曼(Ackmann)函数

## 题目描述

阿克曼 $(Ackmann)$ 函数 $A(m,n)$ 中，$m，n$ 定义域是非负整数 $(m \le 3,n \le 10)$，函数值定义为：

$akm(m,n)=n+1$；($m=0$ 时 )。

$akm(m,n)=akm(m-1,1)$；($m>0,n=0$ 时 )。

$akm(m,n)=akm(m-1,akm(m,n-1))$；（$m,n>0$ 时 )。

## 输入格式

$m$ 和 $n$。

## 输出格式

函数值。

## 样例 #1

### 样例输入 #1
```
2 3
```

### 样例输出 #1

```
9
```
