# 可持久化动态仙人掌的直径问题

## 题目描述

给定 $n,m$，求有多少个正整数 $x$，使得 $x^m\le n$。

## 输入格式

一行两个正整数 $n,m$。

## 输出格式

一个整数表示正整数 $x$ 的个数。

## 样例 #1

### 样例输入 #1
```
5 2
```

### 样例输出 #1

```
2
```

## 提示

对于 $25\%$ 的数据满足 $m=1$；  
对于 $50\%$ 的数据满足 $n\le 10^6$；  
对于 $100\%$ 的数据满足 $1\leq n,m\le 10^9$。
