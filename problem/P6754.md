# [BalticOI 2013 Day1] Palindrome-Free Numbers

## 题目背景



## 题目描述

我们定义回文串为正着读反着读都一样的数字串，如果一个数字串的一个长度大于 $1$ 的子串也为回文串的话，那么我们也定义这个数字串为回文串。

顾名思义，不是回文串的数字串为非回文串。

给定 $a,b$，求区间 $[a,b]$ 中有多少个数字串为非回文串。

## 输入格式

一行两个整数 $a,b$ 代表这个区间。

## 输出格式

一行一个整数代表答案。

## 样例 #1

### 样例输入 #1
```
123 321```

### 样例输出 #1

```
153```

## 样例 #2

### 样例输入 #2
```
123456789 987654321```

### 样例输出 #2

```
167386971```

## 提示

#### 数据规模与约定

对于 $100\%$ 的数据，$0 \le a \le b \le 10^{18}$。    
对于其中的 $25\%$ 的数据，$b-a \le 10^5$。
