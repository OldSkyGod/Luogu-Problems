# [USACO07OPEN]Catch That Cow S

## 题目描述

FJ丢失了他的一头牛，他决定追回他的牛。已知FJ和牛在一条直线上，初始位置分别为x和y，假定牛在原地不动。FJ的行走方式很特别：他每一次可以前进一步、后退一步或者直接走到2\*x的位置。计算他至少需要几步追上他的牛。


## 输入格式

第一行为一个整数t(≤10)，表示数据组数；接下来每行包含一个两个正整数x和y(0<x,y≤10^5)，分别表示FJ和牛的坐标。


## 输出格式

对于每组数据，输出最少步数。


## 样例 #1

### 样例输入 #1
```
1 
5 17
```

### 样例输出 #1

```
4
```
