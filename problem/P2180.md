# 摆石子

## 题目描述

我们伟大的KK在N条水平线与M条竖直线构成的网格中（KK的自创坐标系），放K枚石子，每个石子都只能放在网格的交叉点上。现在KK想知道在最优的摆放方式下，最多可以找到多少四边平行于坐标轴的长方形，而且KK要求它的四个角上都恰好放着一枚石子。


## 输入格式

一行输入三个正整数 N，M，K。


## 输出格式

一行输出一个正整数，表示最多的满足条件的长方形数量。


## 样例 #1

### 样例输入 #1
```
3 3 8
```

### 样例输出 #1

```
5
```

## 提示

50%的数据    0<N，M≤30；

100%的数据   0<N，M≤30000  K≤N\*M。
