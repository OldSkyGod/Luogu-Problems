# [省选联考 2021 B 卷] 数对

## 题目描述

给定 $n$ 个正整数 $a_i$，请你求出有多少个数对 $(i, j)$ 满足 $1 \le i \le n$，$1 \le j \le n$，$i \ne j$ 且 $a_i$ 是 $a_j$ 的倍数。

## 输入格式

第一行，一个整数 $n$，表示数字个数。  
第二行，$n$ 个整数，表示 $a_i$。

## 输出格式

输出一行，一个整数，表示答案。

## 样例 #1

### 样例输入 #1
```
6
16 11 6 1 9 11
```

### 样例输出 #1

```
7
```

## 样例 #2

### 样例输入 #2
```
见附件中的 pair/pair2.in。
```

### 样例输出 #2

```
见附件中的 pair/pair2.ans。
```

## 提示

对于 $40 \%$ 的数据，$n \le 1000$。  
对于 $70 \%$ 的数据，$1 \le a_i \le 5 \times {10}^3$。  
对于 $100 \%$ 的数据，$2 \le n \le 2 \times {10}^5$，$1 \le a_i \le 5 \times {10}^5$。
