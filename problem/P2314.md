## 输入格式

从文件input.txt中读入数据，文件第一行为整数L，其中4<=L<=100000，且有50%的数据满足L<=10^4，表示木板下侧直线段的长。第二行为L个正整数A1,A2,…,AL，其中1<Ai<=10^8，表示第i段的高度。


## 输出格式

输出文件output.txt中仅包含一个整数D，表示为使梳子面积最大，需要从木板上挖掉的格子数。


## 样例 #1

### 样例输入 #1
```
9 
4 4 6 5 4 2 3 3 5
```

### 样例输出 #1

```
3
```
