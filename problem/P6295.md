# 有标号 DAG 计数

## 题目背景



## 题目描述

对 $n$ 个点有标号的有向无环图进行计数，要求：弱连通图（所有的有向边替换为无向边后的图为连通图）。输出答案对 
 $998244353$ 取模的结果。

## 输入格式

一个整数 $T$。

## 输出格式

共 $T$ 行，第 $i(1\le i\le T)$ 行输出 $n=i$ 时的答案。

## 样例 #1

### 样例输入 #1
```
5```

### 样例输出 #1

```
1
2
18
446
26430```

## 提示

第一个点 $T = 2000$。

第二个点 $T = 100000$。
