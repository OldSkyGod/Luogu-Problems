# 进制转换

## 题目描述

请你编一程序实现两种不同进制之间的数据转换。


## 输入格式

共三行，第一行是一个正整数，表示需要转换的数的进制$n(2≤n≤16)$，第二行是一个n进制数，若$n>10$则用大写字母$A-F$表示数码$10-15$，并且该$n$进制数对应的十进制的值不超过$1000000000$，第三行也是一个正整数，表示转换之后的数的进制$m(2≤m≤16)$。


## 输出格式

一个正整数，表示转换之后的$m$进制数。


## 样例 #1

### 样例输入 #1
```
16
FF
2
```

### 样例输出 #1

```
11111111
```
