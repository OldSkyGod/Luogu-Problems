# [CCC2021 S1] Crazy Fencing

## 题目背景



## 题目描述

栅栏由 $N$ 块木板组成。

因为是栅栏，所以第 $i$ 块的右边界与第 $i+1$ 块的左边界的高度一样。

告诉你这些木板的边界的高度与这些木板的宽度，输出这些木板的总面积。

## 输入格式

第一行一个整数 $N$。

第二行 $N+1$ 个整数，表示每个边界的高度。

第三行 $N$ 个整数，表示每块木板的宽度。

## 输出格式

一行一个保留一位小数的实数，总面积。

## 样例 #1

### 样例输入 #1
```
3
2 3 6 2
4 1 1
```

### 样例输出 #1

```
18.5```

## 样例 #2

### 样例输入 #2
```
4
6 4 9 7 3
5 2 4 1
```

### 样例输出 #2

```
75.0```
