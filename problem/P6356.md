# [COCI2007-2008#3] CUDAK

## 题目背景



## 题目描述

求出在区间 $[A,B]$ 中有多少个整数的各位数字之和是 $S$。并请找出最小的那个整数。

## 输入格式

输入一行三个整数 $A,B,S$。



## 输出格式

输出共两行，第一行为整数的个数，第二行为最小的整数。

输入数据保证至少有一个这样的数。

## 样例 #1

### 样例输入 #1
```
1 9 5```

### 样例输出 #1

```
1
5```

## 样例 #2

### 样例输入 #2
```
1 100 10```

### 样例输出 #2

```
9
19```

## 样例 #3

### 样例输入 #3
```
11111 99999 24```

### 样例输出 #3

```
5445
11499```

## 提示

#### 数据规模与约定

对于 $100\%$ 的数据，保证 $1\le A,B\le 10^{15}$，$1\le S\le 135$。
