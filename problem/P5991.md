# [PA2015]Równanie

## 题目描述

对于一个正整数 $n$，定义 $\operatorname{f}(n)$ 为它十进制下每一位数字的平方的和。

现在给定三个正整数 $k,a,b$，请求出满足 $a\le n\le b$ 且 $k\times \operatorname{f}(n)=n$ 的 $n$ 的个数。

## 输入格式

第一行包含三个正整数 $k,a,b$。

## 输出格式

输出一个整数，即满足条件的 $n$ 的个数。

## 样例 #1

### 样例输入 #1
```
51 5000 10000
```

### 样例输出 #1

```
3
```

## 提示

对于 $100\%$ 的数据，$1\le k,a,b\le 10^{18}$，$a\le b$。

---

### 样例解释：

满足的 $3$ 个 $n$ 分别为 $7293,7854$ 和 $7905$。
