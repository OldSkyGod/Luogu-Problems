# [PA2011]Prime prime power 质数的质数次方

## 题目背景



## 题目描述

对于给定的数 $n$，求第 $k$ 小的 $a^b$（$a,b$ 都为质数），使得它的值大于 $n$。

## 输入格式

一行两个整数 $n,k$。

## 输出格式

一行一个整数为答案。

## 样例 #1

### 样例输入 #1
```
22 2```

### 样例输出 #1

```
27```

## 样例 #2

### 样例输入 #2
```
22 1```

### 样例输出 #2

```
25```

## 提示

对于 $100\%$ 的数据，$1\le n\le 10^{18}$，$1\le k\le 10^5$。
