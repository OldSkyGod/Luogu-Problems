# [SCOI2009] windy 数

## 题目背景

windy 定义了一种 windy 数。

## 题目描述

不含前导零且相邻两个数字之差至少为 $2$ 的正整数被称为 windy 数。windy 想知道，在 $a$ 和 $b$ 之间，包括 $a$ 和 $b$ ，总共有多少个 windy 数？

## 输入格式

输入只有一行两个整数，分别表示 $a$ 和 $b$。

## 输出格式

输出一行一个整数表示答案。

## 样例 #1

### 样例输入 #1
```
1 10```

### 样例输出 #1

```
9```

## 样例 #2

### 样例输入 #2
```
25 50
```

### 样例输出 #2

```
20
```

## 提示

#### 数据规模与约定

对于全部的测试点，保证 $1 \leq a \leq b \leq 2 \times 10^9$。
