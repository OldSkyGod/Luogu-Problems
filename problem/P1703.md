# 那个什么密码2

## 题目背景

原题为 https://www.luogu.com.cn/problem/P1079。

## 题目描述

与原题一模一样、具体不同请见输入格式


## 输入格式

第一行输入密钥


第二行输入明文


第三行输入一个正整数 $M$, 代表操作个数.


每个操作将 $[a,b]$ 的字符全部倒转，比如 abc 倒转为 cba.


接下来 $M$ 行，每行输入两个正整数 $a$, $b$。


## 输出格式

输出一行,输出密文。


## 样例 #1

### 样例输入 #1
```
CompleteVictory
Wherethereisawillthereisaway
0```

### 样例输出 #1

```
Yvqgpxaimmklongnzfwpvxmniytm```
