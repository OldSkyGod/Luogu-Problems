# 火星上的加法运算

## 题目描述

最近欢欢看到一本有关火星的书籍，其中她被一个加法运算所困惑，由于她的运算水平有限.她想向你求助，作为一位优秀的程序员，你当然不会拒绝。


## 输入格式

第一行先输入一个运算的进制N(2<=N<=36)，接下来两行为需要进行运算的字符，其中每个字符串的长度不超过200位，其为N进制的数。其中包括0-9及a-z（代表10-35）。


## 输出格式

在N进制下它们的和


## 样例 #1

### 样例输入 #1
```
20
1234567890
abcdefghij
```

### 样例输出 #1

```
bdfi02467j
```
