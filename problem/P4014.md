# 分配问题

## 题目描述

有 $n$ 件工作要分配给 $n$ 个人做。第 $i$ 个人做第 $j$ 件工作产生的效益为 $c_{ij}$ 。试设计一个将 $n$ 件工作分配给 $n$ 个人做的分配方案，使产生的总效益最大。


## 输入格式

文件的第 $1$ 行有 $1$ 个正整数 $n$，表示有 $n$ 件工作要分配给 $n$ 个人做。

接下来的 $n$ 行中，每行有 $n$ 个整数 $c_{ij}$​​，表示第 $i$ 个人做第 $j$ 件工作产生的效益为 $c_{ij}$。


## 输出格式

两行分别输出最小总效益和最大总效益。


## 样例 #1

### 样例输入 #1
```
5
2 2 2 1 2
2 3 1 2 4
2 0 1 1 1
2 3 4 3 3
3 2 1 2 1
```

### 样例输出 #1

```
5
14
```

## 提示

$1 \leq n \leq 100$

一个人只能修一个工件
