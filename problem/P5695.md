# [NOI2001] 反正切函数的应用

## 题目描述

我们将公式 $ 4 $ 写成如下形式

$$ \arctan(\frac{1}{a}) = \arctan(\frac{1}{b}) + \arctan(\frac{1}{c}) $$

其中 $ a, b, c \in \mathbb{N^+} $。

我们的问题是：对于每一个给定的 $a$，求 $ b + c $ 的值。我们保证对于任意的 $ a $ 都存在整数解。如果有多个解，要求你给出 $ b + c $ 最小的解。


## 输入格式

输入文件中只有一个正整数 $ a $。

## 输出格式

输出文件中只有一个整数，为 $ b + c $ 的值。

## 样例 #1

### 样例输入 #1
```
1
```

### 样例输出 #1

```
5
```

## 提示

$1 \le a \le 6\times 10^4 $。
