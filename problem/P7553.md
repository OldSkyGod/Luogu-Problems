# [COCI2020-2021#6] Geometrija

## 题目描述

平面内有不共线的 $n$ 个点。如果两条线段 $\overline{AB}$  和 $\overline{CD}$ 有异于 $A, B, C, D$ 的公共点，则称他们「相交」。

记 $S$ 为 $n$ 个点两两相连得到的线段的集合。求不与 $S$ 中任意其他线段相交的线段数量。

## 输入格式

第一行一个整数 $n$。

接下来 $n$ 行，每行两个整数 $x_i, y_i$，表示第 $i$ 个点的坐标。

## 输出格式

一行一个整数，表示满足要求的线段的数量。

## 样例 #1

### 样例输入 #1
```
4
1 1
-1 1
-1 -1
1 -1
```

### 样例输出 #1

```
4
```

## 样例 #2

### 样例输入 #2
```
4
-1 -1
1 -1
0 1
0 0
```

### 样例输出 #2

```
6
```
