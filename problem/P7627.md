# [COCI2011-2012#1] X3

## 题目描述

给定一个长度为 $N$ 的序列 $A_1,A_2,...,A_N$，求序列元素两两异或的总和。

## 输入格式

输入的第一行包含一个正整数 $N$。

接下来 $N$ 行每行包含一个正整数 $A_i$。

## 输出格式

输出一行一个整数，表示两两异或后的总和。

## 样例 #1

### 样例输入 #1
```
2
19
10
```

### 样例输出 #1

```
25
```

## 样例 #2

### 样例输入 #2
```
3
7
3
5
```

### 样例输出 #2

```
12
```

## 样例 #3

### 样例输入 #3
```
5
9
13
1
9
6
```

### 样例输出 #3

```
84
```

## 提示

#### 【样例 2 解释】

$A_1 ⊕ A_2 = 4$，$A_1 ⊕ A_3 = 2$，$A_2 ⊕ A_3 = 6$，$4 + 2 + 6 = 12$，因此答案为 $12$。

#### 【数据范围】

对于 $100\%$ 的数据，$1 \le N \le 10^6$，$1 \le A_i \le 10^6$。
