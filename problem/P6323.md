# [COCI2006-2007#4] ZBRKA

## 题目背景



## 题目描述

在一个长度为 $n$ 的排列中找出逆序对数量恰好为 $c$ 的排列总数。（$\bmod \ 10^9+7$）

## 输入格式

输入共一行为两个整数 $n,c$。

## 输出格式

输出逆序对数量恰好为 $c$ 的排列总数。（$\bmod \ 10^9+7$）

## 样例 #1

### 样例输入 #1
```
10 1```

### 样例输出 #1

```
9```

## 样例 #2

### 样例输入 #2
```
4 3```

### 样例输出 #2

```
6```

## 样例 #3

### 样例输入 #3
```
9 13```

### 样例输出 #3

```
17957```

## 提示

#### 数据规模与约定

对于 $100\%$ 的数据，保证 $1\le n\le 10^3$，$1\le c\le 10^4$。
