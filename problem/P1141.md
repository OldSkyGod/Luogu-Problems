# 01迷宫

## 题目背景



## 题目描述

有一个仅由数字 $0$ 与 $1$ 组成的 $n \times n$ 格迷宫。若你位于一格 $0$ 上，那么你可以移动到相邻 $4$ 格中的某一格 $1$ 上，同样若你位于一格 $1$ 上，那么你可以移动到相邻 $4$ 格中的某一格 $0$ 上。

你的任务是：对于给定的迷宫，询问从某一格开始能移动到多少个格子（包含自身）。

## 输入格式

有一个仅由数字 $0$ 与 $1$ 组成的 $n \times n$ 格迷宫。若你位于一格 $0$ 上，那么你可以移动到相邻 $4$ 格中的某一格 $1$ 上，同样若你位于一格 $1$ 上，那么你可以移动到相邻 $4$ 格中的某一格 $0$ 上。

你的任务是：对于给定的迷宫，询问从某一格开始能移动到多少个格子（包含自身）。

## 输出格式

$m$ 行，对于每个询问输出相应答案。


## 样例 #1

### 样例输入 #1
```
2 2
01
10
1 1
2 2
```

### 样例输出 #1

```
4
4
```

## 提示

所有格子互相可达。

对于 $20\%$ 的数据，$n \leq 10$；

对于 $40\%$ 的数据，$n \leq 50$；

对于 $50\%$ 的数据，$m \leq 5$；

对于 $60\%$ 的数据，$n,m \leq 100$；

对于 $100\%$ 的数据，$n \leq 1000,m \leq 100000$。
