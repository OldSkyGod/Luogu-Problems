# [CQOI2007]矩形

## 题目描述

给一个a\*b矩形，由a\*b个单位正方形组成。你需要沿着网格线把它分成非空的两部分，每部分所有格子连通，且至少有一个格子在原矩形的边界上。“连通”是指任两个格子都可以通过水平或者竖直路径连在一起。 求方案总数。例如3*2的矩形有15种方案。

![](https://cdn.luogu.com.cn/upload/pic/18502.png)

## 输入格式

输入仅一行，为两个整数a，b。
1<=a<=6, 2<=b<=7

## 输出格式

输出仅一行，即方案总数。

## 样例 #1

### 样例输入 #1
```
3 2
```

### 样例输出 #1

```
15
```
