# [Ynoi2018] GOSICK

## 题目描述

维多利加给了你一个序列 $a$，每次询问给一个区间 $[l,r]$。

查询 $l \leq i,j\leq r$，且 $a_i$ 是 $a_j$ 倍数的二元组 $(i,j)$ 的个数。




## 输入格式

第一行两个整数表示 $n,m$。

第二行 $n$ 个整数表示序列 $a$。

之后 $m$ 行，每行两个整数 $l,r$ 表示一次询问。


## 输出格式

对于每个询问，输出一行一个整数表示答案。

## 样例 #1

### 样例输入 #1
```
6 3
1 1 4 5 1 4
1 1
4 5
1 4
```

### 样例输出 #1

```
1
3
10
```
