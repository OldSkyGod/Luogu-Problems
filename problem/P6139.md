# 【模板】广义后缀自动机（广义 SAM）

## 题目描述

给定 $n$ 个由小写字母组成的字符串 $s_1,s_2\ldots s_n$，求本质不同的子串个数。（不包含空串）

## 输入格式

第一行一个正整数 $n$。

以下 $n$ 行，每行一个字符串，第 $i$ 行表示字符串 $s_{i-1}$。

## 输出格式

一行一个正整数，表示答案。

## 样例 #1

### 样例输入 #1
```
4
aa
ab
bac
caa
```

### 样例输出 #1

```
10
```

## 提示

数据范围：$1\le n\le 4\cdot 10^5$，$1\le \sum{|s_i|}\le 10^6$。

样例解释：共有 $10$ 个本质不同的子串，它们是：`"a","b","c","aa","ab","ac","ba","ca","bac","caa"`。
