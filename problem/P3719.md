# [AHOI2017初中组]rexp

## 题目背景

以下为不影响题意的简化版题目。


## 题目描述

给出一个由(,),|,a组成的序列，求化简后有多少个a。

化简规则：

1、形如aa...a|aa...a|aa...a的，化简结果为“|”两边a的个数最多的一项，例如a|aa|aaa=aaa
3、先算带括号的序列，例如(a|a)|aaa=aaa


## 输入格式

一行一个序列


## 输出格式

化简后a的个数


## 样例 #1

### 样例输入 #1
```
aa(aa)|(aa|(a|aa))aa
```

### 样例输出 #1

```
4
```
