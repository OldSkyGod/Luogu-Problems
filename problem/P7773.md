# [COCI2009-2010#2] RIMSKI

## 题目描述

给定一个罗马数字 $B$，把 $B$ 的字符重新排列，要求让排列后的数字最小。

## 输入格式

一行一个罗马数字 $B$。

## 输出格式

一行一个罗马数字，为你重排后能得到的最小的数字。

## 样例 #1

### 样例输入 #1
```
VIII
```

### 样例输出 #1

```
VIII
```

## 样例 #2

### 样例输入 #2
```
VI
```

### 样例输出 #2

```
IV
```

## 样例 #3

### 样例输入 #3
```
III
```

### 样例输出 #3

```
III
```

## 样例 #4

### 样例输入 #4
```
LI
```

### 样例输出 #4

```
LI
```

## 提示

$1\leq B< 100$。

注意，在本题中`I`如果在大数字之前，它只能在`V` 、`X`之前。输入遵循同样的规则。（这就是样例 $4$ 的输出为什么是 `LI` 而不是 `IL`。）
