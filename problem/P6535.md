# [COCI2013-2014#1] TRENER

## 题目背景

有一些字符串，需要你按特定要求统计。

## 请不要只输出 `PREDAJA`，浪费洛谷评测资源！


## 题目描述

给出 $n$ 个字符串，每个字符串都有其对应首字母。

输出首字母对应字符串数量不低于 $5$ 的首字母个数。

## 输入格式

第一行一个正整数 $n$。

接下来 $n$ 行每行包含一个仅由小写字母组成的字符串，长度不超过 $30$ 个字符。

## 输出格式

若不存在符合题目要求的首字母，输出 `PREDAJA`。

否则按字典序输出所有首字母，其中无空格。

## 提示

#### 【数据规模与约定】
对于 $100 \% $ 的数据，$1\le n \le150$。
