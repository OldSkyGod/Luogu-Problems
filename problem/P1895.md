# 数字序列

## 题目背景

查找数字


## 题目描述

在下列的无穷数字序列1121231234123451234561234567123456781234567891234567 8910123456789101112345678910......中，查找第i个数字。


## 输入格式

第一行为正整数t(1≤t≤10)，表示测试数据组数。接下来t行，每行一个正整数i(1≤i≤2^31-1)。


## 输出格式

对于每一个i，输出对应位置上的数字。


## 样例 #1

### 样例输入 #1
```
2
3
8
```

### 样例输出 #1

```
2
2
```
