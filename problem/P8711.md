# [蓝桥杯 2020 省 B1] 整除序列

## 题目背景



## 题目描述

有一个序列，序列的第一个数是 $n$，后面的每个数是前一个数整除 $2$，请输出这个序列中值为正数的项。

## 输入格式

输入一行包含一个整数 $n$。

## 输出格式

输出一行，包含多个整数，相邻的整数之间用一个空格分隔，表示答案。

## 样例 #1

### 样例输入 #1
```
20```

### 样例输出 #1

```
20 10 5 2 1```

## 提示

对于 $80\%$ 的评测用例，$1\le n\le10^9$。

对于所有评测用例，$1\le n\le10^{18}$。
