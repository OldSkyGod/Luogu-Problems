# 生日礼物

## 题目描述

wyh有一个幸运数字n。现在hke打算送他用心度为a的礼物，ljc打算送他用心度为b的礼物。两个人的礼物合在一起，用心度为lcm(a,b)，其中lcm代表最小公倍数。

如果wyh收到礼物的用心度恰好为n，他就会很高兴。现在请问有多少种送礼物的方案使得wyh高兴？


## 输入格式

一个数n


## 输出格式

方案数


## 样例 #1

### 样例输入 #1
```
6
```

### 样例输出 #1

```
9
```

## 提示

//这9种方案为：(1,6),(2,6),(3,6),(6,6),(2,3),(3,2),(6,1),(6,2),(6,3)

30%的数据，n≤1000；

100%的数据，n≤10^16
