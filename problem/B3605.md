# [图论与代数结构 401] 二分图匹配

## 题目描述

给定一张左侧有 $nl$ 个点、右侧有 $nr$ 个点、$m$ 条边的二分图，求一组它的最大匹配。

## 输入格式

第一行三个整数 $nl$，$nr$，$m$。

接下来 $m$ 行，每行两个整数 $u_i$，$v_i$，表示有一条左侧第 $u_i$ 个点连向右侧第 $v_i$ 个点的边。


## 输出格式

第一行一个整数表示最大匹配数。

## 样例 #1

### 样例输入 #1
```
2 2 3
1 1
1 2
2 1
```

### 样例输出 #1

```
2
```

### 样例输入 #2

### 样例输出 #2
